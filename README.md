# Курсовая работа: модель Лотки-Вольтерры

Репозиторий содержит курсовую работу по математической модели популяционной динамики Лотки-Вольтерры для варианта 33.

## Тема

Модель Лотки-Вольтерры описывает взаимодействие двух популяций: жертв и хищников. В работе выполнены теоретический анализ системы, аналитический поиск точек равновесия, численное моделирование и сравнение результатов.

## Параметры варианта

| Параметр | Значение |
|---|---:|
| alpha | 0.489 |
| beta | 0.03762 |
| delta | 0.00675 |
| gamma | 0.27 |

Начальные условия:

| Опыт | x(0), жертвы | y(0), хищники |
|---:|---:|---:|
| 1 | 28 | 9 |
| 2 | 52 | 10 |
| 3 | 32 | 17 |
| 4 | 48 | 16 |

## Основные результаты

Для системы

```text
dx/dt = alpha*x - beta*x*y
dy/dt = delta*x*y - gamma*y
```

аналитически найдены точки равновесия:

```text
E0 = (0, 0)
E* = (gamma / delta, alpha / beta) = (40, 12.9984)
```

Анализ матрицы Якоби показывает, что точка `E0` является седловой, а положительная стационарная точка `E*` является центром. Численное моделирование подтверждает колебательное движение фазовых траекторий вокруг `E*`.
